- **Always-sync-latest: strictly-increasing client sync clock (sync.ts)** (done, E2E-verified). The sync clock advanced via `writeClock(Date.now())` on every change, which can move the clock BACKWARD ‚Äî below a value the device previously synced/adopted (another device's slightly-ahead clock, device clock-skew, or a future-dated import/seed).
